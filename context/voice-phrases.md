# Voice Phrase Reference

Complete list of Hindi phrases the app recognizes and acts on.

---

## List naming phrases

Any Hindi phrase ≥ 2 characters is accepted as a list name.

| Example voice | List name created |
|---|---|
| मार्च का किराना लिस्ट | मार्च का किराना लिस्ट |
| जनवरी की लिस्ट | जनवरी की लिस्ट |
| घर का राशन | घर का राशन |
| मम्मी की लिस्ट | मम्मी की लिस्ट |

## TTS responses (app speaks these)

| Trigger | App says |
|---|---|
| App opens | "नमस्ते! कृपया अपनी लिस्ट का नाम बोलें" |
| List name set | "ठीक है! [नाम] बनाई गई। अब बोलें क्या चाहिए।" |
| Item added | "ठीक है, लिस्ट में जोड़ दिया।" |
| 6s after item | "और कुछ? आप और कुछ ऐड करना चाहते हैं क्या?" |
| Stop command | "ठीक है, लिस्ट सेव हो गई।" |
| Bad speech | "क्या आप मुझसे कुछ कहना चाहते हैं? कृपया करके दोबारा दोहराएं, मैं आपकी बात को समझ नहीं पायी।" |
| New list | "नई लिस्ट के लिए नाम बोलें" |
| Name too short | "कृपया अपनी लिस्ट का नाम बोलें" |
